# Summer5

This repo is split into two apps:

- client: Vite + React frontend
- server: Node + Express backend

## Setup

```sh
npm i
npm --prefix client i
npm --prefix server i
```

## Run both

```sh
npm run dev
```

## Run individually

```sh
npm run dev:client
npm run dev:server
```

## Notes

Copy `server/.env.example` to `server/.env` and fill in values before starting the server.
